Overview

Built for agents that actually live somewhere.

Most harnesses are one-shot terminal wrappers. Talon runs as a long-lived process with persistent memory, hot-swappable MCP plugins, and frontends for wherever you need it — CLI, Telegram, Microsoft Teams, or your own.

Capabilities

Six things that matter.

01

Claude Agent SDK

First-class Anthropic integration. Adaptive thinking, prompt caching, streaming, tool use, sub-agents.

02

Plugin-based

15+ MCP plugins ship in the box. Drop-in path plugins follow a uniform shape. Hot-reload at runtime.

03

Long-term memory

Mempalace integration. ChromaDB vector store + SQLite knowledge graph. Semantic recall across sessions.

04

Multi-frontend

Same brain, different mouths. Telegram, Teams, terminal — or write your own.

05

Dream & Heartbeat

Background modes for memory consolidation and proactive maintenance. Reflects on its own logs while you sleep.

06

Self-hosted

Your VPS, your data, your model choice. 1,400+ tests. No vendor lock-in beyond the model itself.

Plugins

A growing ecosystem of MCP tools.

Internal plugins live in the talon repo. External path plugins live in the talon-agent org and follow a uniform shape: a package.json + an src/index.ts.

Get started

Up and running in under a minute.

Install, initialise a config, run. Bring your own Anthropic key.

# install
$ npm install -g talon-agent

# initialise
$ talon init

# run
$ talon start --frontend terminal
