Talon is a self-hosted, plugin-based agent harness for Claude. Multi-frontend. Long-term memory. Lives where you live.

Get started → View on GitHub
$ npm install -g talon-agent
// what it is

Built for agents that actually live somewhere.

Most harnesses are one-shot terminal wrappers. Talon runs as a long-lived process with persistent memory, hot-swappable MCP plugins, and frontends for wherever you want it — CLI, Telegram, Microsoft Teams, or your own.

⚡

Built on Claude Agent SDK

First-class Anthropic integration. Adaptive thinking, prompt caching, streaming, tool use, sub-agents — it all just works.

🔌

Plugin-based

15+ MCP plugins ship in the box. Drop-in path plugins follow a uniform shape. Hot-reload at runtime, no restart.

🧠

Long-term memory

Mempalace integration: ChromaDB vector store + SQLite knowledge graph. Semantic recall across sessions, not just a context window.

💬

Multi-frontend

Same brain, different mouths. Run as a Telegram bot, a Teams bot, a terminal app — pick or write your own.

🌙

Dream & Heartbeat

Background modes for memory consolidation and proactive maintenance. The agent reflects on its own logs while you sleep.

🦅

Self-hosted

Your VPS, your data, your model choice. No vendor lock-in beyond the model itself. 1,400+ tests keep it honest.

// the ecosystem

Plugins that ship in the box.

// get started

Up and running in 60 seconds.

Install, point at a config file, run. Bring your own Anthropic key.

~/talon 
# Install globally
$ npm install -g talon-agent

# Initialize a config
$ talon init

# Run with the terminal frontend
$ talon start --frontend terminal

# Or as a Telegram bot
$ talon start --frontend telegram
